# Кооперативная многозадачность
Реализация обхода графа в глубину с использованием кооперативной многозадачности

# Решение

## Сборка, запуск

```bash
make 
```

## Очистка

```bash
make clean
```

## Результат

Граф выглядит следующим образом:
```
 1 — 3 — 5 - 6
 | \   \ |
 0 — 2 — 4 
```

1. Task1 (DFS_1): 0
2. Task2 (DFS_2): 3
3. Task3 (DFS_3): 4
